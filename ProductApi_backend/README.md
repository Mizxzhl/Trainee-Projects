# 🚀 Product API (ASP.NET Core)

A backend API currently under development, focused on building a **real-world e-commerce system** using clean architecture and best practices.

## 🔥 Current Progress

* CRUD operations for Products, Categories, Prices
* Customer & Address modules
* Cart system with backend price calculation
* DTO separation (Create / Update / Response)
* Service layer for business logic
* Generic Repository pattern
* Entity Framework Core integration

## 🛠️ Tech Stack

* ASP.NET Core Web API
* Entity Framework Core
* SQL Server
* C#

## 📌 Architecture

* Controller → Service → Repository pattern
* Clean separation between Entities and DTOs
* Async/await for database operations
