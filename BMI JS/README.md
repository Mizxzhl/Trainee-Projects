# BMI Calculator

A simple web app to calculate Body Mass Index (BMI).

## Features
- Input weight and height
- Calculate BMI instantly
- Display result on screen

## Tech Used
- HTML
- JavaScript

## How it works
- User enters weight and height through input prompts
- BMI is calculated using the formula
- Result is displayed on the page


## Purpose
Built to practice basic calculations, user input handling, and JavaScript functions.
